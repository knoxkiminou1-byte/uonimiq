## Packages
zustand | Global state management for the cart drawer and UI state
framer-motion | Cinematic page transitions, scroll reveals, and micro-interactions
clsx | Utility for conditional class joining
tailwind-merge | Utility to merge tailwind classes safely

## Notes
- Dark theme is forced as the default to match the "Obsidian/Cinematic" fashion house aesthetic.
- Hero video is expected to be served from `/videos/hero.mp4`.
- Zod is used for runtime validation of all API responses.
- Cart state is managed client-side via Zustand for a snappy drawer experience.
